# BuildsWar

Protótipo web de um RPG assíncrono centrado em builds, loot e relatórios de combate.

## Estado atual

- interface responsiva inspirada em portais clássicos de MMORPG;
- três builds demonstrativas;
- equipamentos, habilidades e dungeon demonstrativa;
- árvore passiva interativa com 18 nós;
- três caminhos de Ascendência;
- bônus de talentos refletidos nos atributos;
- motor determinístico e persistência ainda pendentes.

## Executar localmente

```bash
npm install
npm run dev
```

## Verificações

```bash
npm run typecheck
npm test
npm run build
```

O planejamento do protótipo está em [docs/PROTOTYPE_PLAN.md](./docs/PROTOTYPE_PLAN.md).
